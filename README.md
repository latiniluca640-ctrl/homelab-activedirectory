# Home Lab - Active Directory

Lab domestico per lo studio pratico della sistemistica IT 
e della sicurezza informatica, in preparazione al ruolo di 
Sistemista IT e al percorso verso SOC Analyst.

## Obiettivo
Costruire un ambiente reale e documentato che dimostri 
competenze pratiche su Active Directory, hardening, 
logging e analisi degli eventi di sicurezza.

## Ambiente
- Hypervisor: VirtualBox
- OS: Windows Server 2022 Standard Evaluation
- Ruolo principale: Domain Controller (lab.local)
